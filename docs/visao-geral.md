# Visão geral

O objetivo do sistema é apoiar o monitoramento de compromissos decenais da criança e do adolescente e dos problemas públicos associados permitindo buscas, filtragem e visualização hierárquica de problemas. Um problema público contém um compromisso decenal e que, por sua vez, possui um objetivo. Cada objetivo pode conter múltiplas linhas de ação vigiadas por uma secretaria responsável e por colaboradores. Além disso, cada linha de ação contém múltiplas tarefas as quais podem possuir vários indicadores associados. Além da visualização, o sistema permite o gerenciamento dos problemas públicos e dos demais dados associados, desde que os usuários tenham permissões necessárias. O sistema também oferece como função permitir a visualização detalhada de dados sobre os indicadores que são atualizados periodicamente.
